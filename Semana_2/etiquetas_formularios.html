Etiqueta Descripcion Ejemplo de uso
<input type="text"> define un campo de entrada de texto de una sola línea



<input type="password"> define un campo de contraseña



<input type="submit"> define un botón para enviar datos de formulario a un administrador de formulario





<input type="reset"> define un botón de reinicio que reiniciará todos los valores del formulario a sus valores predeterminados





<input type="radio"> define un botón de radio


<input type="checkbox"> define una casilla de verificación


<input type="button"> define un botón
<input type="color"> se utiliza para campos de entrada que deben contener un color.
<input type="date"> se utiliza para campos de entrada que deben contener una fecha
<input type="datetime-local"> especifica un campo de entrada de fecha y hora, sin zona horaria
<input type="email"> se utiliza para campos de entrada que deben contener una dirección de correo electrónico
<input type="file"> se utiliza para campos de entrada que deben contener una dirección de correo electrónico
<input type="number"> define un campo de entrada numérico
<input type="range"> define un control para ingresar un número cuyo valor exacto no es importante (como un control deslizante)
<input type="search"> se usa para campos de búsqueda (un campo de búsqueda se comporta como un campo de texto normal)
<input type="tel"> se utiliza para campos de entrada que deben contener un número de teléfono
<;input type="url"> se utiliza para campos de entrada que deben contener una dirección URL
<input type="time"> permite al usuario seleccionar una hora (sin zona horaria)
<input type="week"> Permite al usuario seleccionar una semana y año.
<Select> representa un control que permite al usuario elegir de entre un número de opciones

Idioma:

<extarea> representa un control para la edición mutilínea de texto sin formato
